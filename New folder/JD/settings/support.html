User Guide

This user manual is designed to provide guidance on using the Gruba Law Library Management System. It covers all essential aspects, from installation to troubleshooting, helping users efficiently manage library resources, track borrowers, and maintain records.

  • Operating System: macOS 10.13 (High Sierra) or later
  • Processor: 64-bit Intel processor or Apple Silicon (M1/M2)
  • RAM: Minimum 4 GB (8 GB or more recommended)
  • Storage: 500 MB of available disk space (1 GB recommended)
  • Display Resolution: Minimum 1280x800 (1920x1080 or higher recommended)

  1. Launch the application from your desktop.
  2. On the login screen, enter the default credentials.
    • Username: admin
    • Password: password
  3. Click the "Login" button. If the credentials are correct, you will be directed to the main dashboard.
  4. If the credentials are incorrect, refer to the password hint for a clue to your password.
  5. If you still cannot recall the password, click "Reset Password" to reset it. You will be prompted to answer a security question. Once answered correctly, you can set a new password.

The dashboard serves as the central interface of the application, providing quick access to essential features.

  • Adding Books: It is the + icon that adds new books directly from the dashboard.
  • Total Number of Books: Displays the total count of books in the library. Clickable: Acts as a shortcut to the "Books" menu for managing book records.
  • Total Number of Borrowed Books: Shows the total count of books currently borrowed. Clickable: Serves as a shortcut to the "Borrower Logs" menu for viewing and managing borrowing activities.
  • Book Statuses: Visual representation of book availability using a pie chart (e.g., available, borrowed, archived).
  • Generating Reports: Access options to generate detailed library activity reports (e.g., availability of books, list of books, borrowing details).
  • Notification: Alerts for overdue borrowed books that have not yet been returned.
  • Recently Added Books: Displays a list of the most recently added books for quick reference.

The sidebar menu allows for seamless navigation across the system. It includes the following sections:

  • Dashboard: Redirects to the main interface of the application, providing an overview of key metrics and features.
  • Books: Access the complete library of books. Includes options to add, edit, delete, or archive books.
  • Borrowers Profile: Manage borrower details, including adding new borrowers and viewing existing profiles.
  • Borrowers Log: View and manage borrowing and returning transactions. Includes sorting, filtering, and search functionality.
  • Settings: Access system configurations, including database management, and account settings.
  • Logout: Securely exit the system.

Adding New Books
  1. Select the "Add Book" option from the dashboard.
  2. Enter book details (e.g., title, author, and publication year). Note: Title and the number of copies are mandatory fields.
  3. Click "Add Book" to save the record in the database.
Viewing Available Books
  1. Navigate to the "Books" section using the sidebar menu.
  2. Browse the displayed list of available books.
  3. To view specific books, use the search bar to locate titles, authors, or other keywords. You can select a category from the dropdown menu to narrow your search, and apply the date filter by the date they were received.
  4. To manage table columns, click the eye icon to toggle between different book views and display additional categories. Use the checkboxes to show or hide specific columns by unchecking or checking them. For quick adjustments, select "Show All" to display all columns or "Hide All" to hide them.
Deleting Books
  1. Locate the specific book in the "Books" section.
  2. Click the "Delete" button next to the book record.
  3. Confirm the action in the prompt.
    ⚠️ Note: Deleted records cannot be recovered. Archive books if you may need them later.
Archiving Books
  1. Locate the book in the "Books" section.
  2. Click the "Archive" button to move the record to the "Archived Books" section.
  3. Archived books can be restored or permanently deleted as needed.

Borrowing Books
  1. Open the "Borrow Logs" section.
  2. Click the "+" icon to add a new record.
  3. Enter the borrower's details (e.g., Borrower ID). Borrowers can borrow up to three books in a single transaction.
  4. Confirm the details and click "Borrow" to complete the transaction.
Viewing Borrower Logs
  1. Navigate to the "Borrower Logs" section.
  2. Use the search bar or sorting options (ascending/descending) to organize records.
  3. Click on the borrower’s name to view their borrowing history.
  4. Click on a book title to view its borrowing history.
  5. Filter records by status for easier navigation.
Returning Books
  1. Locate the borrowed record in the "Borrower Logs" section.
  2. Use the dropdown menu to select "Returned."
  3. The librarian will review the book's condition and update its status accordingly.

Backing Up the Database
  1. Go to the "Settings" section.
  2. Select "Database Management."
  3. Follow the prompts to save a backup copy to your preferred location.
Restoring the Database
  1. Navigate to the "Database Management" section in "Settings."
  2. Select "Restore Database" and follow the prompts to restore the data.
Exporting and Importing Excel Files
  1. In "Settings," choose "Database Management."
  2. To export records, click "Export to Excel" to save book records and borrower profiles to your computer.
  3. To import records, click "Import from Excel" to upload or update the database from an Excel file.
Changing Credentials
  1. Go to "Account Settings" in "Settings."
  2. To change your username, click "Change Username," enter your new username and current password, and click "Save."
  3. To change your password, click "Change Password," enter your current password, a new password, and a password hint. Click "Update Password" to save changes.
Setting Security Questions
  1. In "Account Settings," select "Add Security Question."
  2. Choose from predefined questions or create a personalized question.
  3. Provide a specific and unique answer, then enter your current password to save.

This section provides solutions to common issues encountered while using the Gruba Law Library Management System.

General Troubleshooting
  • Refresh the App: Click reload or Ctrl+R to refresh/reload the app. This can resolve minor display issues or updates not reflecting.
  • Restart the App: Close the application completely. Reopen it and check if the issue is resolved. Restarting can clear temporary glitches or errors.
Login Issues
  • Problem: Unable to log in.
  • Solution: Ensure your username and password are correct. Use the password hint feature if you forget your password. If still unable to log in, use the "Reset Password" option. Answer the security question to reset your password.
  • Problem: System freezes on the login screen.
  • Solution: Restart the application. Ensure your system meets the minimum requirements (refer to the System Requirements section).
Book or Borrower Records Not Saving
  • Problem: Unable to add a new book or borrower record.
  • Solution: Confirm all mandatory fields are filled (e.g., book title, borrower ID). Ensure the database is not full or corrupted. If issues persist, back up and restore the database.
Export/Import Errors
  • Problem: Cannot export or import records to/from Excel.
  • Solution: Verify that the Excel file is not corrupted and follows the required format. Ensure the database management settings are correctly configured.
Backup or Restore Fails
  • Problem: Unable to back up or restore the database.
  • Solution: Ensure sufficient storage space is available. Verify the file path for backup or restore is accessible.
System Crashes or Freezes
  • Problem: The application crashes or becomes unresponsive.
  • Solution: Restart the application and check if your device meets the minimum system requirements. Check for available updates and install them if necessary.
Data Display Issues
  • Problem: Records are not displaying correctly or appear incomplete.
  • Solution: Use the search and filter options to refine the displayed data. Restart the application or check the database for corruption if the issue persists.
Borrow/Return Functionality Not Working
  • Problem: Unable to borrow or return books.
  • Solution: Confirm that the book or borrower records exist in the database. For borrowing, check if the book's status is available. For returns, ensure the dropdown menu is updated and that a librarian has reviewed the book's condition.
Settings and Account Management Errors
  • Problem: Unable to change credentials or add security questions.
  • Solution: Ensure the current password is entered correctly. For security questions, make sure the answer is specific and unique.

If issues persist after following these steps, refer to the Support section for further assistance.
