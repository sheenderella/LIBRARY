User Guide and FAQs

User Guide

Enter your username and password on the login screen. After successfully logging in, you'll be redirected to the dashboard.

If you receive an "Invalid credentials" message, double-check your username and password. If the problem persists, ensure your Caps Lock is off and try again.

Click on the "Add Book" button in the main dashboard. Fill in the book details, including the title, author, and other required fields, then click "Save."

If you encounter an error while saving, ensure all required fields are filled out correctly. If the error persists, restart the app and try again.

Go to the "Borrow Books" section, select the book you want to borrow, and enter the borrower's information. Confirm the details to complete the borrowing process.

If the book is not available for borrowing, ensure it hasn't already been borrowed. If an error occurs, check that the borrower's information is correctly entered.

In the "Borrow Records" section, find the record you wish to update. Click "Edit," make the necessary changes, and then click "Update" to save.

If the record fails to update, ensure all fields are correctly filled. If the error continues, restart the app and attempt the update again.

Locate the borrow record in the "Borrow Records" section, click "Return," and confirm the return to update the record.

If you can't return the book, check if the borrow record is correct and the book hasn't already been marked as returned.

Find the book you want to delete in the "Books" section, click the "Delete" button, and confirm the action. The book will be removed from the system.

If an error occurs while deleting, ensure that the book is not currently borrowed. If it is, return the book first before deleting it.

Go to "Settings," select "Backup," and follow the prompts to create a backup of your library database. The backup file will be saved in the specified location.

If the backup fails, ensure you have write permissions for the save location. If you encounter persistent issues, check that there’s enough disk space and try again.

In the "Settings" section, select "Restore," choose the backup file, and follow the prompts to restore your database from the backup.

If the restore process fails, ensure the backup file is not corrupted and that it matches the format required by the app. Verify that you have proper permissions to access the file.

FAQs

On the login screen, enter your username and password. If your credentials are correct, you will be directed to the main dashboard.

If you get locked out after multiple failed attempts, you may need to reset your password or contact the system administrator.

Click the "Add Book" button. Enter the book's details such as the title, author, and publication year. After filling in all the information, click "Save" to add the book to the library.

If the book doesn't save, check for missing or incorrectly formatted fields. Restart the app if necessary and re-enter the details.

Navigate to the "Borrow Books" section. Select the book you wish to borrow and provide the borrower's information. Confirm to finalize the borrowing process.

If the borrowing process fails, ensure that the book is available and that the borrower's details are correct. If the problem persists, verify that your database is working properly.

In the "Borrow Records" section, find the record that needs updating. Click on "Edit," modify the necessary fields, and click "Update" to save your changes.

If the update doesn’t go through, make sure the fields aren’t empty and that the record exists. Reload the app if issues continue.

To return a book, locate the borrow record, click "Return," and confirm the action. The system will update the record to show the book has been returned.

If the return fails, verify that the book was correctly borrowed under the same record. Restart the app if necessary.

Go to the "Books" section, find the book you want to delete, and click the "Delete" button. Confirm the deletion to remove the book from the library.

If deletion is unsuccessful, ensure no active borrow records are linked to the book. Clear any dependencies before attempting to delete again.

In the "Settings" menu, click "Backup." Follow the instructions to save a backup of your database.

Ensure the backup location is accessible and that you have enough storage space. If backup fails, consider saving to a different location.

Go to "Settings" and select "Restore." Choose the backup file and follow the steps to restore your database.

If the restore fails, verify that the backup file is intact and compatible with your current database schema. Try restoring from an earlier backup if needed.

Navigate to the "Settings" section, select "Change Credentials," and enter your current password along with your new username and/or password. Save the changes to update your login credentials.

If credentials do not update, confirm that the current password is correct and that the new username isn’t already taken. Retry the process if needed.
